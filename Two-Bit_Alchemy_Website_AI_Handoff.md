# Two-Bit Alchemy / Fisher Aquatics Website Transition Report

## Purpose

AI handoff document summarizing the design direction, current status,
and recommended next steps.

## Current Status

-   Existing WordPress site has been untouched for about one year.
-   Before redesign:
    -   Backup site.
    -   Update WordPress core.
    -   Update themes/plugins.
    -   Verify compatibility.
    -   Review security.
    -   Remove \~1,700 spam comments using an appropriate plugin.
    -   Clean obsolete content.

## Strategic Direction

The project evolved from redesigning Fisher Aquatics into creating a
broader personal identity called **Two-Bit Alchemy**.

Fisher Aquatics should become a project within the broader site rather
than the entire identity.

## Visual Identity

Victorian natural-history engraving aesthetic.

Influences: - scientific plates - field journals - cabinet of
curiosities - antique laboratories

Avoid: - corporate look - fantasy magic - steampunk - horror - excessive
animation

Mood: Curious, grounded, handcrafted, quietly mystical.

## Preferred Emblem

Includes: - Triple Moon - Draco constellation - Polaris - microscope -
antique camera - axolotl - botanical elements - Appalachian stream -
candle - notebook - mortar & pestle - subtle pentacle - exposed roots

Future refinements: - Make Draco an authentic celestial engraving. -
Reduce remaining geometric diagrams. - Remove or soften constellation
labels.

### About

Personal rather than résumé-like.

Explain that Two-Bit Alchemy represents transformation through
observation, craftsmanship, patience, and curiosity.

## UX

-   Excellent typography
-   Minimal animation

-   Strong photography and illustration

## Codex Workflow

When available: - Audit WordPress. - Recommend plugins/theme. - Build
reusable templates. - Assist with CSS. - Optimize performance and
accessibility. - Maintain an implementation log after each coding
session.

## User Preferences

-   Planning before coding.
-   Iterative refinement.
-   Authenticity over flash.
-   Concise documentation suitable for AI handoffs.

## Guiding Principle

The website should answer: 'What has Dada been curious about lately?'
rather than: 'What services does this business sell?'
