# Assets

Use this folder for project assets.

## Folders

- `brand/` - logo, emblem, color, typography, and identity files.
- `images/` - general site images.
- `photography/` - original or curated photography intended for site use.
- `references/` - inspiration and non-production reference material.

Do not optimize, rename, or overwrite original assets without recording the change in `logs/implementation-log.md`.
