# Project Summary

## Purpose

Two-Bit Alchemy is the broader personal website and identity that will contain Fisher Aquatics and other ongoing projects, field notes, workshop posts, collections, and reflections.

The guiding question from the handoff is:

> What has Dada been curious about lately?

This is more important than presenting the site as a business brochure.

## Current Project State

- The existing Fisher Aquatics WordPress site has been mostly untouched for about one year.
- WordPress should be backed up, updated, cleaned, and audited before redesign work touches the live site.
- The new site direction is broader than Fisher Aquatics.
- The handoff includes a preferred emblem image and a defined visual language.
- No implementation framework or theme decision has been approved yet.

## Preserved Decisions

- Keep Two-Bit Alchemy as the parent identity.
- Treat Fisher Aquatics as one project area, not the whole brand.
- Preserve the Victorian natural-history engraving aesthetic.
- Favor calm reading, excellent typography, strong imagery, and minimal animation.
- Avoid corporate styling, fantasy magic, steampunk, horror, and excessive motion.
- Keep the site personal, exploratory, and authentic.
- Use `docs/brand-manifest.md` as the long-term creative and design constitution.

## Primary Site Areas

- Home
- Projects
- Workshop Journal
- Field Notes
- Cabinet
- About
- Philosophy
