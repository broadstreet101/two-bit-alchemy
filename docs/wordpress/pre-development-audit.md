# WordPress Pre-Development Audit

Before implementation touches the live site, the existing WordPress installation should be audited and protected.

## Audit Checklist

- Confirm backup process and restore path.
- Update WordPress core.
- Update active theme and plugins.
- Verify compatibility after updates.
- Review security posture.
- Remove spam comments with an appropriate plugin or moderation workflow.
- Clean obsolete content after review.
- Inventory active plugins.
- Inventory custom CSS, theme edits, and widgets.
- Export or document current content structure.
- Check image library size and quality.
- Review performance and accessibility baselines.
